# 🚀 Polygon Deployment (Assignment-2)

## 🔹 Objective

Deploy the smart contract on Polygon Testnet (Mumbai) using Hardhat.

---

## 🔹 Steps to Deploy on Polygon Testnet

1. Install dependencies:
   npm install

2. Configure Hardhat for Polygon Mumbai:

   * Add RPC URL and private key in `hardhat.config.js`

3. Compile the contract:
   npx hardhat compile

4. Deploy the contract:
   npx hardhat run scripts/deploy.js --network mumbai

5. After deployment, contract address will be displayed in terminal
